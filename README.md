# TicTacToe

![CI](https://github.com/alok/TicTacToe/actions/workflows/ci.yml/badge.svg)

A simple TicTacToe game implemented in Lean 4.

## Building

Ensure you have the Lean 4 toolchain installed (see `lean-toolchain`).

```
lake build
```
